ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЯ

Данная программа предназначена для обновления содержимого таблиц в документах Microsoft Word (.docx).
Она заменяет текст в определённых ячейках таблицы на слово "Транспорт" в соответствии с номерами из текстового файла.

ШАГИ ДЛЯ ИСПОЛЬЗОВАНИЯ ПРОГРАММЫ:

1. Подготовка файла с номерами:
   - Создайте текстовый файл с именем "numbers.txt".
   - В этом файле укажите номера, которые должны быть заменены, каждый номер на отдельной строке.
   - Пример содержимого файла:
     ```
     1.5
     4.33
     6.46
     19.1
     69.0
     ```

2. Запуск программы:
   - Запустите программу.
   - Откроется диалоговое окно, в котором нужно выбрать Word документ (.docx).

3. Обработка документа:
   - Программа найдет и заменит текст в ячейках таблицы на "Транспорт" для номеров, указанных в файле "numbers.txt".
   - После завершения работы программа создаст новый файл с добавленным в название " с транспортом".

4. Проверка результатов:
   - Проверьте новый файл, чтобы убедиться, что замены были выполнены корректно.
   - Пример названия нового файла: "Документ с транспортом.docx".

5. Сообщения:
   - Программа выводит сообщения о том, какие номера были найдены и заменены, а какие — нет.

Если у вас возникнут вопросы или проблемы, обратитесь к разработчику программы.
